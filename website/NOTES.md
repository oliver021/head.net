# Website Development Notes

## Pending Features

### OpenAPI Interactive Docs
**Status:** Disabled (pending investigation)  
**Why:** The `docusaurus-plugin-openapi-docs` plugin was causing build failures in GitHub Actions CI environments.

**What we have:**
- OpenAPI spec file: `static/openapi.yaml` — fully describes the Invoice CRUD API
- Plugin packages installed: `docusaurus-plugin-openapi-docs`, `docusaurus-theme-openapi-docs`

**The issue:**
- Build passes locally but fails in CI with truncated errors
- Generated API doc files reference theme components that weren't available
- Autogenerated sidebar integration had conflicts

**To re-enable later:**
1. Uncomment `plugins` array in `docusaurus.config.ts` (lines 18-33)
2. Uncomment `themes` in `docusaurus.config.ts` (line 49)
3. Run locally: `npx docusaurus gen-api-docs all`
4. Test build: `npm run build`
5. Update `sidebars.ts` to include the generated API Reference category
6. Verify CI passes before pushing

**Reference:**
- Plugin docs: https://github.com/PaloAltoNetworks/docusaurus-plugin-openapi-docs
- OpenAPI spec format: `static/openapi.yaml`
